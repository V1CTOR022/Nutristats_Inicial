# 🥗 Nutri Stats - Landing Page

Este projeto é uma **Landing Page** desenvolvida para apresentar a proposta da plataforma **Nutri Stats**, focada em nutrição, acompanhamento alimentar e análise de dados nutricionais.

## 🚀 Tecnologias utilizadas

* **HTML5** → Estrutura da página
* **CSS3** → Estilização e responsividade
* **JavaScript** → Interatividade e funcionalidades dinâmicas

## 💡 Sobre o projeto

A landing page foi criada com o objetivo de:

* Apresentar a ideia do Nutri Stats
* Destacar os benefícios da plataforma
* Oferecer uma interface simples, moderna e intuitiva
* Servir como porta de entrada para futuros recursos (dashboard, análise alimentar, etc.)

## 🎯 Funcionalidades

* Layout responsivo (adaptado para diferentes dispositivos)
* Seções informativas sobre o sistema
* Design limpo e focado na experiência do usuário
* Interações básicas com JavaScript
