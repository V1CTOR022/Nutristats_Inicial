Desenvolvedor & Criador

Olá, eu sou o Victor Yuri

O NutriStats nasceu da minha vontade de monitorar meus gastos calóricos. Como desenvolvedor web, percebi que faltava uma ferramenta que fosse não apenas funcional, mas intuitiva e bonita de se usar no dia a dia.

Meu objetivo com este projeto é entregar uma plataforma completa que ajude as pessoas a transformarem seus hábitos alimentares através de dados reais e acompanhamento preciso.

Tecnologias utilizadas no projeto:

HTML5 CSS3 JavaScript Vanilla SQL
